# 英语如何刷真题

## 1，刷前

1.1，刷提前先行背诵重点词汇

1.2，掌握阅读做题策略

## 2，刷中

2.1，不限时，此阶段重点提升阅读能力

2.2，重视选项，做题时认真分析每个选项，而不是只看正确选项

2.3，标记选项理解状态

这一步骤的目的为找出自己的不足，总结教训

同时帮助自己理解阅读能力的提升状况

## 3，刷后

3.1，再做一遍题目，重做选项里面有不清楚或者完全不理解的题目

3.2，看解析

3.3，整理文章知识点

1. 词，认识全部单词
2. 句，理解全部句子，把握大致结构（判定方法：能否刘畅地默念句子）
3. 篇，明确段落作用，整理文章结构
4. 选项

## 4，及时复习
